# Goal Horn Audio Files

Drop team goal horn MP3s here named by NHL team abbreviation:

```
ANA.mp3   ARI.mp3   BOS.mp3   BUF.mp3   CAR.mp3   CBJ.mp3
CGY.mp3   CHI.mp3   COL.mp3   DAL.mp3   DET.mp3   EDM.mp3
FLA.mp3   LAK.mp3   MIN.mp3   MTL.mp3   NJD.mp3   NSH.mp3
NYI.mp3   NYR.mp3   OTT.mp3   PHI.mp3   PIT.mp3   SEA.mp3
SJS.mp3   STL.mp3   TBL.mp3   TOR.mp3   UTA.mp3   VAN.mp3
VGK.mp3   WSH.mp3   WPG.mp3
```

**`default.mp3` is required** — plays when no team-specific file exists.

The two files already in /public (dallas-goal.mp3, generic-goal.mp3) can be
renamed and moved here:
  cp ../dallas-goal.mp3 DAL.mp3
  cp ../generic-goal.mp3 default.mp3
